📌 Project: Nostra – E-Commerce Fashion Website

🧑‍💻 This project is a responsive fashion e-commerce website built using HTML, Tailwind CSS, and JavaScript.
It showcases a modern online clothing store with product browsing, filtering, and interactive UI components.

✨ Features

• 🏠 Responsive Homepage with hero banner and promotional section

• 🧭 Navigation Bar with links to different sections of the website

• 📱 Mobile Side Navigation Menu for small screens

• 🖼 Hero Image Slider with previous and next navigation buttons

• 🏷 Brand Showcase Section displaying popular clothing brands

• ⭐ Customer Experience Section highlighting service benefits

• 🆕 New Arrivals Section with product preview cards

• 🔥 Most Wanted Products Carousel with horizontal scrolling

• ❤️ Wishlist Icon Interaction for product liking

• 🛍 Collections Page with product listing

• 🔍 Search Functionality to find products quickly

• 🎨 Product Filtering System by:

Category (Formals, Casuals, T-Shirts)

Colors (White, Blue, Black, Yellow)

Arrivals (New / Old)

• 🎯 Product Cards showing image, name, and price

• 📧 Newsletter Subscription Section for updates and offers

• 📦 Limited Offer Banner for promotional sales

• 📱 Fully Responsive Design for mobile, tablet, and desktop screens

🛠 Technologies Used

• HTML5 – Website structure and layout

• Tailwind CSS – Styling, responsiveness, and UI design

• JavaScript – Interactive features like:

Image slider

Side navigation menu

Product filtering

Wishlist icon interaction

Search functionality

⚙️ How it Works

1️⃣ Users visit the homepage and explore featured products

2️⃣ Navigation bar and side menu allow movement between pages

3️⃣ The hero section displays promotional banners with slider controls

4️⃣ Users can view products in New Arrivals and Most Wanted sections

5️⃣ On the Collections page, users can:

Search for products

Filter products by category, color, and arrivals

6️⃣ JavaScript dynamically shows or hides products based on selected filters

7️⃣ Users can subscribe to the newsletter for future offers

🎯 Purpose of the Project

This project was created to practice and demonstrate:

• Responsive web design using Tailwind CSS

• JavaScript DOM manipulation

• Interactive UI development

• Product filtering and search functionality

• Building a real-world e-commerce website layout
